# Makewell Industries — Design System

## Aesthetic: Minimal Posh

Warm stone neutrals, rich charcoal, monochrome CTAs — inspired by Apple-level restraint with industrial gravitas.

| Token | Hex | Usage |
|-------|-----|-------|
| Background | `#FAFAF8` | Page canvas |
| Surface | `#FFFFFF` | Cards, forms |
| Stone | `#F3F2EE` | Alternate sections |
| Foreground | `#121212` | Headlines, CTAs |
| Charcoal | `#0D0D0D` | Dark sections, footer |
| Muted | `#6E6E6E` | Body copy |
| Accent warm | `#9A8B7A` | Labels, subtle highlights |
| Accent line | `#C4B5A3` | Dividers |

## Typography (unchanged)

- **Display:** Syne 800
- **Body:** DM Sans

## Motion (Framer Motion)

- **Hero:** Scroll parallax on image, fade on headline
- **FadeIn:** Elements reveal on scroll
- **Stagger:** Product cards, features, industries cascade in
- **Parallax:** Mission image, lifestyle section scale
- **Header:** Glass blur on scroll, animated dropdown

## Run locally

```bash
cd web && npm run dev
```
